Notes

HTML — core concepts

What is HTML? HTML (HyperText Markup Language) is the standard language for creating web pages. It uses elements (tags) like <div>, <p>, <a> to structure content.

Basic structure — every HTML document has a doctype and <html>, <head>, <body> sections. Example:

<!doctype html>
<html>
  <head>...
  </head>
  <body>...
  </body>
</html>

Semantic HTML: Use tags that describe meaning — <header>, <main>, <article>, <nav>, <footer>. This helps accessibility and SEO.

CSS — styling & layout

What is CSS? Cascading Style Sheets control the look & layout of HTML. You can select elements with selectors, create layout with Flexbox/Grid, and animate with transitions and keyframes.

Box model: Every element has content, padding, border, margin. Use box-sizing: border-box to make sizing predictable.

Layout basics:

  • Flexbox — one-dimensional layout. Use display:flex, justify-content, align-items.
  • Grid — two-dimensional layout. Use display:grid, grid-template-columns.

Responsive design: Use media queries (@media) and relative units (%, rem, vw) to adapt the layout.

JavaScript — behavior & interactivity

What is JavaScript? JS is the programming language of the web used to change HTML/CSS at runtime, handle events, fetch data, and more.

DOM basics: Use document.querySelector, addEventListener, and manipulate element properties like textContent, classList, style.

Common patterns:

  • Event-driven — code runs when user interacts (click, input, submit).
  • State — keep small state objects (like user's quiz answers) and re-render UI from that state.
  • LocalStorage — persist small user data across sessions with localStorage.setItem and .getItem.

Accessibility & best practices

Always use semantic tags, provide alt for images, label form controls (<label>), ensure keyboard navigation and good contrast. Write small, focused functions and keep CSS modular.
